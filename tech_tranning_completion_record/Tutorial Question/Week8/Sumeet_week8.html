1. Provide a definition of Reactive Forms:

Reactive forms is an approach to where you can handle inputs to the form whose values are constantly changing over time.

2. List Reactive Form Classes with its functionality

FormControl is for individual form control, tracks value and validations status FormGroup: groups the FormControl instances AbstractControl : abstract base class for the form control classes FormArray: an indexed array for the AbstractControl instances

3. List Form Control Properties with its Description

setValue() : assign every form control value at once patchValue() : update a specific form control value

4. Provide a definition of RxJS

RxJS is a library that can be used to compose asynchronous and event based programs by using the observable sequences.

5. Provide a definition The Observer Pattern

Yhe observer pattern revolves around observables, these can be tracked and observed by observers. The observers monitor and watch changes that are made to the observables. When an observer notifies a change in state, it notifies all the other observers of the change.
