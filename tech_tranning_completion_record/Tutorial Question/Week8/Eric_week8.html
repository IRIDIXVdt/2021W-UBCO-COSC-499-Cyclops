1. Provide a definition of Reactive Forms:

Explicit management of data flowing between an non-UI data model and a UI-oriented form model

2. List Reactive Form Classes with its functionality

  1. FormControl: powers individual form control, tracks the value and validation status
  2. FormGroup: group of FormControl instances
  3. AbstractControl: abstract base class for the form control classses
  4. FormArray: numerically indexed array of AbstractControl instances

3. List Form Control Properties with its Description

  1. .value: value of the form control
  2. .status : validity: VALID, INVALID, PENDING, DISABLED
  3. .pristine/.dirty: true if user has not changed/changed value in UI
  4. .untouched/: true if the user has not yet entered the HTML
  5. ..touched: control and triggered its blur event

4. Provide a definition of RxJS

Library for composing asynchronous and event-based programs by using observable sequences.

5. Provide a definition The Observer Pattern

  1. Software-engineering pattern
  2. Observable (subject) tracks observers
  3. Observable notifies observers of state changes
