1. Provide a definition of Reactive Forms:

We explicitly manage the data flow between non-UI data model and a UI-oriented form model. This means creating a tree of angular form control objects within the component class instead of putting all form controls in the HTML template. This tree is binded to elements in the template file.

2. List Reactive Form Classes with its functionality

  1. FormControl: individual form controls, tracks form values and validation status
  2. FormGroup: group of FormControls
  3. AbstractControl: abstract base class for the form control classes
  4. FormArray: numerically indexed array of AbstractControl instances

3. List Form Control Properties with its Description

for checking form control values

  1. .value: value
  2. .status: valid, invalid, pending, disabled
  3. .pristine/.dirty: true if user did not change the value/true if value changed
  4. true if user has not interacted with control/true if user has interacted with control

4. Provide a definition of RxJS

a library that angular uses to provide a reactive way of writing angular applications, used for composing asynchronous and event-based programs by using observable sequences

5. Provide a definition The Observer Pattern

The observer pattern is a software-engineering pattern. this is where there are observables, which are many different streams, observers can subscribe to an observeable, which is when they can so it's notified of any changes of that observeable .
